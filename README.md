# Monkeying Around  

![Splash Art](/Splash Art.png)  

Swing from tree trunk to tree trunk in this thrilling action non-adventure. You won't go anywhere doing this, but it might be fun!  

## Controls  
- **WASD** or **Arrow Keys** – Move  
- **Space** – Grab
